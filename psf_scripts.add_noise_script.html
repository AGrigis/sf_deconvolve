psf_scripts.add_noise_script module¶

ADD NOISE SCRIPT

This script does the following: - Reads in “clean” images and PSF(s). - Convolves the PSF(s) with the images. - Adds Gaussian noise to the convolved images. - Results are saved to a numpy binary file.
